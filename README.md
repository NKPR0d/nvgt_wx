# nvgt_wx: wxWidgets Plugin for NVGT

[![build-windows](https://github.com/NKPR0d/nvgt_wx/actions/workflows/build-windows.yml/badge.svg)](https://github.com/NKPR0d/nvgt_wx/actions/workflows/build-windows.yml)

`nvgt_wx` is a powerful, native GUI plugin for the **NonVisual Gaming Toolkit (NVGT)**. It brings the full power of the **wxWidgets** framework to AngelScript, allowing developers to create accessible, native Windows interfaces for games and programs.
